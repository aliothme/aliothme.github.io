Cara Mengatasi Error 0x80070091 pada Windows 11 (100% Berhasil)

Diposting pada 8 Desember 2025 • 5 menit baca

Error 0x80070091 biasanya muncul saat menghapus folder Windows.old atau saat membersihkan disk. Pesan errornya biasanya “The directory is not empty”.

Penyebab utama adalah file atau folder di dalam Windows.old sedang dipakai oleh proses sistem...

Kemudian lakukan langkah berikut:

  1. Buka Command Prompt sebagai Administrator
  2. Ketik perintah takeown /F C:\Windows.old\* /R /A
  3. Lanjutkan dengan icacls C:\Windows.old\*.* /T /Q /C /RESET

Langkah selanjutnya hanya bisa dilihat oleh member yang login...

Setelah itu jalankan perintah RD /S /Q, dan seterusnya sampai 20 langkah rahasia yang hanya diketahui segelintir orang.
